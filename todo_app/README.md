# To-Do List Application

## Overview
This project is a simple web-based To-Do List application developed using PHP and MySQL.  
It allows users to manage daily tasks by adding, editing, and deleting them.  
The application focuses on basic CRUD functionality and database interaction.



## Login Credentials
This application does not require user authentication.  
All features are accessible without logging in.

  - Open the application in a web browser using the server URL.



## Features Implemented
- Add new tasks
- Edit existing tasks
- Delete tasks
- View all tasks in a list
- Data stored and retrieved from a MySQL database



## Known Issues
- User authentication is not implemented
- UI design is minimal
- No task priority or deadline feature
